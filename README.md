## 🎯 Fitur Utama

### 1. **Homepage**
- Navigasi utama website
- Hero section dengan deskripsi brand
- Showcase produk unggulan
- Call-to-action untuk order

- Daftar lengkap produk kopi
- Varian:
  - Manglayang Arabica
  - Manglayang House Blend
  - Premium Honey Process

### 4. **Hubungi Kami**

- Informasi lokasi
- Detail kontak perusahaan

### 5. **Lokasi Toko**
- Informasi tentang toko/kafe fisik
- Jam operasional

## 🎨 Design & Technology

- **HTML5** - Struktur markup
- **CSS3** - Styling dan responsive design
- **Tema Warna:**
  - Primary Green: `#2d5a27` (hijau forest premium)
  - Dark Olive: `#1c2e1a` (untuk teks)
  - Background Cream: `#fbf9f5` (latar belakang lembut)
- **Responsive:** Dioptimalkan untuk desktop, tablet, dan mobile

## 🚀 Cara Membuka

1. Clone repository ini atau download file
2. Buka file `index.html` di browser Anda
3. Gunakan navigasi untuk menjelajahi halaman lain

### Catatan:
- Pastikan semua file HTML berada dalam satu folder dengan folder `aset/`
- Gunakan path relatif untuk gambar agar dapat ditampilkan dengan benar
